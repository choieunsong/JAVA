# 문제 정의

1. N*M 직사각형 모양의 연구소에 빈 칸(0), 벽(1), 바이러스(2)가 있다.
2. 바이러스는 상하좌우로 퍼진다.
3. 연구소에 딱 3개의 벽을 설치해서 빈칸(0)의 수를 최대가 되게 했을 때, 안전영역의 최댓값을 구하시오
4. 3 ≤ N, M ≤ 8, 2 ≤ 2의개수 ≤ 10

# 문제 풀이

1. 칸의 개수가 최대 64이고 선택할 벽의 수가 3개이므로 완전탐색 64C3으로 풀 수 있다.
2. 0인 칸 중에서 3개를 고른다
3. 바이러스를 상하좌우로 퍼트린다.
4. 남은 빈칸의 개수를 세고 최대값과 비교 갱신한다.

간단한 bfs 문제였다.
