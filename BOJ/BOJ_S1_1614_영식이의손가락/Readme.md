# 문제 정의

1. 엄지 → 검지 → 중지 → 약지 → 새끼 → 약지 → 중지 → 검지 → 엄지 순서로 수를 센다.
2. 엄지부터 1,2,3,4,5로 숫자가 매겨져 있으며 영식이는 다친 손가락으로 N번 만큼만 셀 수 있다.
3. 셀 수 있는 최대값을 출력하시오. 1 ≤ N ≤ 1,000,000,000

# 문제 풀이

1: 0 + 8 * N

2: 1 + 8 * (N/2) + (N%2)*6

3: 2 + 8 * (N/2) + (N%2)*4

4: 3 + 8 * (N/2) + (N%2)*2

5: 4 + 8 * N

주의할 점은 N과 결과값이 Int값을 넘을 수 있기 때문에 long 타입으로 선언해 주어야 한다.
