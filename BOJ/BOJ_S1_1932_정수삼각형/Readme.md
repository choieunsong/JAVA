# 문제 정의

1. 정수 삼각형 n(1≤n≤500)이 주어진다. 각 수는 모두 정수이며 범위는 0 이상 9999이하
2. 맨 위층부터 시작해서 아래에 있는 수 중 하나를 선택할 때 이제까지 선택된 수의 합이 최대가 되는 수를 구하여라. 현재 층에서 선택된 수의 대각선 왼쪽 또는 오른쪽만 선택 가능

# 문제 풀이

1. 단순한 dp문제이다.
2. 가장 왼쪽이거나 오른쪽일 떈 바로 위에 있는 값 + 현재 수를 dp에 저장
3. 아닐 땐 max(dp[i][j-1], dp[i][j]) + 현재값을 저장
4. 마지막엔 가장 밑 행에 최대값들이 저장된다. 그 중 최대값을 찾아서 리턴
