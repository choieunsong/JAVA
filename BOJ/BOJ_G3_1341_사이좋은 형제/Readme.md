# 문제 정의

1. 영식이가 케이크의 1/2를 먹으면 민식이는 남은 절반(1/4)를 먹는다
2. "영식이 민식이 영식이" 처럼 패턴을 만들 수 있다.
3. 영식이가 먹게되는 전체 케이크 양이 5/7 이렇게 주어질 때 영식이, 민식이 패턴을 출력하시오.(영식이는 *, 민식이는 -)
4. a, b는 서로소, 1≤ b ≤  2^63-1, 0 ≤ a ≤ b

# 문제 풀이

1. 영식이와 민식이는 케이크의 절반씩만 먹을 수 있으니 모든 분수의 분모는 2의 지수승이다
2. 이걸 이진수로 표현할 수 있다. 만약 영식이, 민식이 순으로 케이크를 먹으면 영식이가 먹은 전체 케잌의 합은 101010...이 되고 민식이는 그 반대인 010101..이 된다
3. 영식이가 먹은 케이크가 분수로 나타났으니 분수를 이진수로 변환해주면 된다.
4. 소수의 이진수 표현은 아래와 같다. `(분자 * 2) / 분모`

```java
2/3 
1. 2*2/3 => 4/3 => 4 > 3 => 1
2. 1*2/3 => 2/3 => 2 < 3 => 0
3. 2*2/3 => 4/3 => 4 > 3 => 1
....
```

1. 처음 예외 처리를 해준다. a가 0이면 무조건 0000이므로 민식이가 전부 먹는다. "-" 출력하고 종료
2. a == b이면 무조건 1111이므로 영식이가 전부 먹는다. "*" 출력하고 종료
3. 패턴 길이가 최대 60까지기 때문에 120까지 문자열을 만든다. 이 때 `a == 0이 되면 이후 무조건 분자는 0이 되므로 패턴이 없어진다.` 이 때 -1 출력하고 종료해야 한다!
4. 패턴을 만든 후에 패턴 길이를 1부터 60까지 일일이 반복되는지 체크해서 반복되면 바로 출력하고 종료한다. 
5. 만약 8번에서 종료되지 않으면 패턴이 없는 것이므로 -1 출력한다

# 정리

이걸 분수의 이진표현으로 푸는 생각을 못했다. 진짜 개빡치네. 그리고 문자열 만들때 분자가 0인 경우 예외처리를 안해주니까 2%에서 계속 틀렸습니다가 떠서 이틀 헤맸다.  수학이 너무 부족한 것 같다.
